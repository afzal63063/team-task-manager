# 🚀 Team Task Manager

A full-stack web application to manage team projects and tasks with role-based access control.

---

## 🔗 Live Demo

- 🌐 https://teamtaskmanager.up.railway.app/login
---

## 🛠 Tech Stack

### Frontend
- React.js
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)

### Deployment
- Railway

---

## ✨ Features

- 🔐 User Authentication (Login / Signup)
- 🧑‍💼 Role-based Access (Admin / User)
- 📁 Create Projects (Admin only)
- 📝 Create & Assign Tasks
- 👥 Assign Tasks to Users
- 📊 View Tasks & Projects
- 🔒 Secure APIs with JWT Authentication
- ☁️ Fully Deployed
